# Simple Project Management App

This repository contains a minimal project management web application inspired by Easy Redmine. It is implemented with Flask and SQLAlchemy using SQLite for storage.

## Features
- Create and list projects
- Add tasks to a project
- Mark tasks as completed

## Setup
Install dependencies and run the server:

```bash
pip install -r requirements.txt
python src/app.py
```

## Running tests

```bash
pytest -q
```
